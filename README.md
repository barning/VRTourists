# VRTourists
Dieses Projekt befasst sich mit der Frage wie Touristen, die einen Überblick über einen ö entlichen Raum wie einem Marktplatz haben möchten, Informationen erhalten und gleichzeitig nicht vollkommen vom Medium eingenommen werden.VR Tourist ist eine App, die Touristen so eine schnelle Orientierung bietet. Mit Hilfe von Technologien, die normalerweise in Virtual Reality Apps zu  nden sind, kann sich der Besucher Informationen auf seinem Tablet anzeigen lassen.Das Tablet wird wie eine Kamera gehalten. Der Tourist sieht einen simplen virtuellen Raum auf dem Bildschirm, durch den er durch bloßes Umschauen mit dem Tablet navigieren kann. Durch Tippen auf eingeblendete Elemente werden erklärende Texte oder Audio- kommentar abgerufen.Durch diese universelle Bedienungsweise ist der Einsatz der App nicht nur für den Außenbereich sinnvoll, sondern kann bei Aus- stellungen oder Veranstaltungen genutzt werden, bei denen es schwierig oder zu aufwendig ist, eine Führung anzubieten.
